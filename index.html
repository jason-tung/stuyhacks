What is Groot-Calculator?

Groot-Calculator is a Python based calculator which takes inputs using HTML Form Processing. That's all.
It takes either a set of ordered pairs or a frequency list
First input however many lines you think you'll need for your list
Then input the the number of lists you used & the data you want
You'll get some cool-ish data on the info
